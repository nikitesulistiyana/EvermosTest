How to Run Automation:
1. You can use any editor, in this case I use PyCharm 
2. To start project just create "New Project" and all library is there
3. Go to Test Folder and open the script "LoginAndChangeStoreName.robot"
4. How to run the script is already exist inside the script "Document", but you can also use this command in Terminal -> robot Tests/LoginAndChangeStoreName.robot
5. Result test is on the folder "Result" just open using any browser
6. DONE
